SukiUI is a modern UI theme library for [AvaloniaUI](https://avaloniaui.net/) that gives your desktop applications a polished, animated look out of the box. It provides GPU-accelerated backgrounds, smooth theme switching, glassmorphism components, and a rich set of styled controls — all designed for desktop use.

## Why SukiUI?

SukiUI brings a cohesive, modern design language to Avalonia desktop apps without requiring you to style every control manually. Key highlights:

- **Light & Dark themes** — Switch between modes at runtime with a single API call
- **GPU-accelerated backgrounds** — Animated shader-based backgrounds (Bubble, Gradient, Waves, and more) using SKSL
- **Glassmorphism design** — GlassCard and related components give your UI a layered, translucent feel
- **Toast & Dialog system** — MVVM-friendly overlays that work from any layer of your application
- **Rich animated controls** — Progress bars, steppers, loading spinners, and more with smooth animations

## Get started in 3 steps

<Steps>
  <Step title="Install the NuGet package">
    Add SukiUI to your Avalonia project:
    ```bash
    dotnet add package SukiUI --version 6.0.0
    ```
  </Step>
  <Step title="Add SukiTheme to App.axaml">
    Replace your existing theme with `SukiTheme`:
    ```xml App.axaml
    <Application xmlns:sukiUi="clr-namespace:SukiUI;assembly=SukiUI">
      <Application.Styles>
        <sukiUi:SukiTheme ThemeColor="Blue" />
      </Application.Styles>
    </Application>
    ```
  </Step>
  <Step title="Use SukiWindow as your main window">
    Replace `Window` with `SukiWindow` in your `MainWindow.axaml`:
    ```xml MainWindow.axaml
    <sukiUi:SukiWindow
        x:Class="MyApp.MainWindow"
        xmlns="https://github.com/avaloniaui"
        xmlns:sukiUi="clr-namespace:SukiUI.Controls;assembly=SukiUI">
    </sukiUi:SukiWindow>
    ```
  </Step>
</Steps>
