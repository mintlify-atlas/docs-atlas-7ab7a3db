title: "DataGrid"
description: "SukiUI styles the AvaloniaUI DataGrid with themed column headers, cell borders that match the SukiUI border brush, and optional cell alignment classes."
---

SukiUI provides a custom `DataGridStyle.axaml` that overrides the column header and cell templates. The grid integrates with the active SukiUI theme — header backgrounds use `SukiLightBorderBrush`, separators use `SukiControlBorderBrush`, and selected rows pick up the primary color.

<Note>
The AvaloniaUI `DataGrid` control lives in the `Avalonia.Controls.DataGrid` NuGet package. Make sure it is referenced in your project and that you have called `UseDataGrid()` (or included the relevant styles) in your `App.axaml`.
</Note>

## Basic usage

```xml
<DataGrid ItemsSource="{Binding Items}"
          AutoGenerateColumns="False"
          CanUserResizeColumns="True"
          GridLinesVisibility="All">
    <DataGrid.Columns>
        <DataGridTextColumn Header="Name"
                            Binding="{Binding Name}"
                            Width="*" />
        <DataGridTextColumn Header="Status"
                            Binding="{Binding Status}"
                            Width="Auto" />
        <DataGridTextColumn Header="Value"
                            Binding="{Binding Value}"
                            Width="120" />
    </DataGrid.Columns>
</DataGrid>
```

## Auto-generated columns

When `AutoGenerateColumns="True"` is set, the DataGrid generates one `DataGridTextColumn` per public property of the bound item type. SukiUI's header and cell styles are applied automatically.

```xml
<DataGrid ItemsSource="{Binding Items}"
          AutoGenerateColumns="True"
          CanUserResizeColumns="True" />
```

## Cell alignment classes

SukiUI exposes two CSS classes on `DataGridCell` for aligning cell content without a custom `CellTemplate`:

| Class | Alignment |
|---|---|
| *(default)* | Left-aligned |
| `CenterAlign` | Horizontally centered |
| `RightAlign` | Right-aligned with a 15 px right margin |

Apply alignment classes via the column's `CellStyleClasses` property:

```xml
<DataGridTextColumn Header="Amount"
                    Binding="{Binding Amount}"
                    Width="100">
    <DataGridTextColumn.CellStyleClasses>
        <x:String>RightAlign</x:String>
    </DataGridTextColumn.CellStyleClasses>
</DataGridTextColumn>
```

## Column headers

Column headers are styled with:

- `SukiLightBorderBrush` background
- `SukiControlBorderBrush` bottom border (1.5 px)
- Semi-bold font weight (`DefaultDemiBold`)
- 12 px padding on all sides

Headers support click-to-sort when `CanUserSortColumns="True"` (the default). The sort indicator arrow is rendered inside the header template.

## Row selection

Row selection uses standard AvaloniaUI `DataGrid` properties:

```xml
<DataGrid ItemsSource="{Binding Items}"
          SelectedItem="{Binding SelectedItem}"
          SelectionMode="Single">
    ...
</DataGrid>
```

## Read-only grid

```xml
<DataGrid ItemsSource="{Binding Items}"
          IsReadOnly="True"
          AutoGenerateColumns="True" />
```
